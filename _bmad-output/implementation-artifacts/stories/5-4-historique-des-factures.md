# Story 5.4: Historique des factures

## User Story

**As a** gestionnaire (building manager),
**I want to** view invoice history and status,
**So that** I can track payments and resolve billing issues.

## Acceptance Criteria

**Given** Invoices have been generated and sent
**When** I access the billing dashboard
**Then** I see all invoices with status, amounts, and payment info
**And** I can filter and export invoice data

## Technical Requirements

### Functional Requirements
- **REQ-WEB-009**: Historique des factures envoyées avec statuts
- **REQ-WEB-011**: Export des données au format CSV/Excel

### Implementation Notes
- Invoice history dashboard
- Status tracking and filtering
- Payment status integration
- Data export functionality
